# cat

打字跟随

桌面端系统，支持 windows 与 macos，具体是用 Electron 实现的一个暹罗猫（我家小猫）同步跟随键盘每一个按键打字，同时张嘴吐小花花，每隔 5s 会眨眼一次，可拖拽修改位置。欢迎下载体验!
